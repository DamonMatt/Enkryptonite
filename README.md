# Enkryptonite

Porta & DES Algorithms

### Prerequisites
- Python3

### Usage
to encrypt:
```
enkryptonite.py [ des | porta ] --message "Hello World!" --key "0123456789ABCDEF" 
```
to decrypt:
```
enkryptonite.py [ des | porta ] --cipher "1f797e16614dab0a65050c8c69d1909d" --key "0123456789ABCDEF" -d
```
